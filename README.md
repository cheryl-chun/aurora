## 特点

| 能力 | 说明 |
| --- | --- |
| 划词翻译 | 在其他应用中选中文本后，用全局快捷键快速弹出翻译结果。 |
| 追加翻译 | 遇到论文跨页、跨段落无法一次选中的文本时，可以继续追加选区并重新翻译完整内容。 |
| 自主管理 API | 支持在应用内配置、启用和排序自己的翻译 API。 |
| 轻量占用 | release 版本后台运行时，当前实测常驻内存约 101-103 MB。 |

## 适合场景

- 阅读英文论文时快速翻译术语、句子或跨页段落。
- 浏览网页、PDF、文档时临时查看译文。
- 使用自己的 API 服务统一管理翻译能力。
